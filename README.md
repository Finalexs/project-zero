# Project Zero

Project Zero is an AI Company OS.

Instead of building complicated workflows, users can build an AI company made of digital employees that research, write, review, and execute work.

## Vision

The goal is to let businesses hire AI employees that can work together, remember company context, and safely use tools to complete real tasks.

## Current MVP

The current version includes:

- Landing page
- AI employee preview
- Command input mockup
- Live company dashboard preview
- Employee marketplace preview
- Pricing section
- Waitlist section

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- GitHub

## Status

Project Zero is currently in early MVP development.
